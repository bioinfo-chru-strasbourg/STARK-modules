SIFT v.1: Submit Protein Sequence

This is an OLDER version of SIFT and is NOT maintained.
It is still made available for those working on similar bioinformatics algorithms.
SIFT v.2 should be use for most prediction purposes.

You can also submit your protein sequence and related sequences or aligned sequences if you already have them.
The SIFT procedure is described in Genome Research 11: 863-874.

When to change this option

Account for Gaps
If the alignment turns out to be especially gapped, resubmit with this option turned off.

Remove sequences more than percent identical to query
This cleans out any polymorphic alleles (sequences in the database that are nearly identical to your protein but already containing the substitutions of interest). The contaminating sequences may cause the substitution to be predicted as tolerated.
