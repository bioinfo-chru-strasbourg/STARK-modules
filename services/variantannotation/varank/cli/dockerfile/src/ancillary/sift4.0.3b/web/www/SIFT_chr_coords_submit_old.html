SIFT Genome Analysis

SIFT Genome


SIFT Genome provides SIFT predictions for a list of chromosome positions and alleles classified as SNPs. This tool is based on NCBI 36 assembly of the human genome.

Please provide your email address and input your data by using the 'upload file' option if input size is greater than 1000 chromosome locations. To avoid overloading our resources, please restrict your input file size to be smaller than 1M. In order to do this we advice that the input may be filtered to retain chromosome locations only for coding exons , as SIFT does not predict for non-coding locations.

Results are deleted after an hour, so please save them!
