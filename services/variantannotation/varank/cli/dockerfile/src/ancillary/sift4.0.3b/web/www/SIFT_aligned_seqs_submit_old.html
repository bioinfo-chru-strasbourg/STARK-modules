SIFT: Submit aligned sequences


Submit your query sequence along with other protein sequences that are aligned to it for SIFT. Your query sequence is the sequence that you would like prediction on (i.e. the sequence in which you have introduced the amino acid substitutions). The query sequence should be first in the alignment. The positions of amino acid substitutions should be with respect to the query sequence. Partial sequences should be flanked by Xes at the beginning and end of the sequence so that those positions are not considered gaps (and hence have liberal prediction).

Parameters

Remove sequences more than percent identical to query
This cleans out any polymorphic alleles (sequences in the alignment that are nearly identical to your protein but already containing the substitutions of interest). The contaminating sequences may cause the substitution to be predicted as tolerated.
