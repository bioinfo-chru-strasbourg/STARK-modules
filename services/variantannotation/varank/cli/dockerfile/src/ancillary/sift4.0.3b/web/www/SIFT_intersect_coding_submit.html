SIFT Genome -> Coding

Please use this page prior to SIFT indels or SIFT Genome to trim your variant list to variants located only in coding regions.
To be used if you have >25,000 SNPs or >1,000 indels.


Input: List of Genome Coordinates
Output: Smaller List of Genome Coordinates in Coding Regions


Results are deleted after an hour, so please save them!
